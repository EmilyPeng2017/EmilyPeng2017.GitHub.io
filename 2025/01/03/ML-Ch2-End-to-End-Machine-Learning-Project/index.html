ML-Ch2-End to End Machine Learning Project

Introduction

In this chapter, I followed the instructions and work through an example project end to end.

Main Steps:

  1. look at the big picture
  2. get the data
  3. explore and visualize the data to gain insights
  4. prepare the data for machine learning algorithms
  5. select a model and train it
  6. fine-tune the model
  7. present the solution
  8. launch, monitor, and maintain the system.

look at the big picture

Task: to use California census data to build a model of housing prices in the state.
Data: it includes metrics such as the population, median income, and median housing price for each block group in California.

Frame the problem

  • supervised learning
  • regression task -> multiple regression & univariate regression
  • batch learning

Select a performance measure

A typical performance measure for regression problems: root mean square error (RMSE)

$$
\begin{equation}
\text{RMSE}(\mathbf{X}, \mathit{h}) = \sqrt{\frac{1}{m} \sum_{i=1}^m \left(h(\mathbf{x}^{(i)}) - y^{(i)}\right)^2}
\end{equation}
$$
